🔥 Firebase Setup

Follow these steps to connect BizPro to your Firebase project.

1 Create Firebase Project

Go to console.firebase.google.com → Create Project → Enable Google Analytics (optional) → Continue.

2 Enable Authentication

In Firebase Console → Authentication → Get Started → Enable Email/Password sign-in method.

3 Create Firestore Database

Firestore Database → Create Database → Start in test mode → Choose region → Enable.

4 Enable Storage

Storage → Get Started → Start in test mode → Done.

5 Get Config Keys

Project Settings (gear icon) → General → Your Apps → Add Web App → Register → Copy the firebaseConfig object.

6 Paste Config Below

Paste your firebaseConfig values and click Save & Launch.

7 Firestore Security Rules

Paste these rules in Firestore → Rules tab:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; allow read: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } match /{collection}/{docId} { allow read, write: if request.auth != null && resource.data.userId == request.auth.uid; allow read, write: if request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } } }

8 Deploy to Firebase Hosting

npm install -g firebase-tools firebase login firebase init hosting # Select your project, set public dir to ".", SPA: yes firebase deploy
